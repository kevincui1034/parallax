How it works
The orchestration is the product.
⚡
Agent that acts, not narrates
A vision LLM emits validated JSON actions — explode, highlight, isolate, focus, reset — that the frontend executes live.
🧩
Real part separation
PartCrafter outputs parts in one canonical frame, so exploded views are physically coherent — not a single fused mesh.
🛡️
Graceful fallbacks
Single-mesh model → fixture → typed input. The contract shape never changes, so there's always something explorable.
